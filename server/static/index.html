Help — inline tags
Put these on their own line inside Text (for the text formats):
  • @#@divider="-=" — repeats the pattern to fill the line width. The pattern can be any characters (=, *~, -·-…). @#@devider is accepted too.
  • @#@cats — inserts a random simple ASCII cat.
More divider ideas: ASCII dividers gallery · more cat art: ASCII cats gallery.
The bundled cats are simple generic ones; paste any art from the galleries straight into Text if you want a specific piece.
